Program 21: Factorial Calculation

Aim: Create a class `Factorial` with a method that accepts a number and returns its factorial in a package `P1`. Using the `Factorial` class, write a program to find the factorial of a number.

package javaproject;

public class Factorial {
    public int calculateFactorial(int num) {
        int result = 1;
        for (int i = 1; i <= num; i++) {
            result *= i;
        }
        return result;
    }
}

// Main application
package javaproject;

import java.util.Scanner;

public class Mainapp {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Enter a number to find its factorial: ");
        int number = scanner.nextInt();
        Factorial factorial = new Factorial();
        int result = factorial.calculateFactorial(number);
        System.out.println("Factorial of " + number + " is: " + result);
        scanner.close();
    }
}

Output:

Enter a number to find its factorial:
3
Factorial of 3 is: 6

Program 22: Handle Arithmetic Exception

Aim: Write a program to handle arithmetic exceptions.

package javaproject;

import java.util.Scanner;

public class ArthematicExc {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter the numerator: ");
        int numerator = scanner.nextInt();
        System.out.print("Enter the denominator: ");
        int denominator = scanner.nextInt();
        try {
            int result = numerator / denominator;
            System.out.println("Result: " + result);
        } catch (ArithmeticException e) {
            System.out.println("Error: Cannot divide by zero");
        }
    }
}

Output:

Enter the numerator: 6
Enter the denominator: 0
Error: Cannot divide by zero

Enter the numerator: 6
Enter the denominator: 3
Result: 2

Program 23: Custom Exception

Aim: Create a user-defined exception `OddValExp` to be invoked when the read number is an odd number.

package javaproject;

public class OddValExce extends Exception {
    public OddValExce(String message) {
        super(message);
    }
}

// Main application
package javaproject;

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter the number: ");
        int number = scanner.nextInt();
        try {
            if (number % 2 != 0) {
                throw new OddValExce("Error: The number is odd!");
            }
            System.out.println("The number is even: " + number);
        } catch (OddValExce e) {
            System.out.println(e.getMessage());
        }
        scanner.close();
    }
}

Output:

Enter the number: 9
Error: The number is odd!

Enter the number: 12
The number is even: 12
